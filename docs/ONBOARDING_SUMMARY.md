### 10 Personas
1. Compliance Officer

8. AI Product Manager

### Onboarding Flow
1. **Persona Selection** (Required) - Choose from 10 role cards
2. **Knowledge Assessment** (Optional) - 3-5 scenario-based multiple choice questions
3. **Interest Selection** (Optional) - Multi-select cards (e.g., Regulatory Compliance, Ethical AI)
4. **Goal Selection** (Optional) - Multi-select cards (e.g., AIGP Certification, Career Advancement)
5. **Completion** - Profile summary + Startegizer introduction

### Key Features
- **Card-based UI** - Simple, visual selection
- **Skippable steps** - But warned that Startegizer won't be personalized
- **Knowledge scoring** - Beginner/Intermediate/Advanced based on answers
- **Profile completion** - Required for full Startegizer experience

### Startegizer Integration
- **Premium feature** - Paid access required
- **Profile-based prompts** - Filtered by persona, knowledge level, interests, goals
- **Personalized experience** - Different prompt library for each user profile
- **Unlock criteria** - Premium subscription + complete profile
